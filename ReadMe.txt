2 - Juggle Fest

Directions to Run the program 
--Place input.txt and output.txt in a folder and run ReadWriteFile.java

Description 

-- ReadWriteFile.java is the main file that parses the contents of input.txt and triggers ArrangeJugglers.java Program 
-- ArrangeJugglers.java assigns all Jugglers to all Circuits and Write Solution to output.txt .
-- Juggler.java file contains Desciprtion of a juggler entity
-- Circuit.java file contains description of a circuit entity
-- Skills.java Describes Skills of Juggler or required Skill of Circuit (Contains hand to eye coordination (H), endurance (E) and pizzazz (P) values)
